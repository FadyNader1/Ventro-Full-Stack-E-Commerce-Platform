## ‚ú® Features

- **Standalone Architecture:** Built entirely using Angular Standalone Components for better performance and smaller bundle sizes.
- **State Management:** Efficient data handling using **RxJS** and **Angular Signals**.
- **Auth System:** Secure Login/Register flow, including **Social Auth (Google & Facebook)** integration.
- **Dynamic Shopping Cart:** Real-time cart management powered by **Redis** (via Backend) and Local Storage.
- **Product Discovery:** Advanced filtering by category, price sorting, and live search.
- **Wishlist:** Personalized wishlist for authenticated users.
- **Checkout & Payments:** Multi-step checkout process integrated with **Stripe**.
- **User Dashboard:** Order history tracking and profile management.
- **Responsive UI:** Fully mobile-first design using **Tailwind CSS** / **Bootstrap**.
